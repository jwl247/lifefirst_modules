TROUBLESHOOTING PATHS
======================

If something fails, check these in order:

1. Apache Status
   └─ sudo systemctl status apache2

2. MySQL Status
   └─ sudo systemctl status mysql

3. Apache Error Log
   └─ sudo tail -f /var/log/apache2/error.log

4. File Permissions
   └─ ls -la /var/www/html/lifefirst/

5. Database Tables
   └─ mysql -u root -p
      └─ SHOW TABLES;

6. Firewall Rules
   └─ sudo ufw status

7. Network Connectivity
   └─ ping YOUR_SERVER_IP
